# 📝 Task Management System (CLI-Based)

A simple 🐍 Python-based Task Management System that helps users manage daily tasks using a command-line interface 💻.  
This project demonstrates core Python concepts like file handling, data structures, and JSON storage 💾.

---

## ✨ Features
- ➕ Add new tasks with unique IDs  
- 📋 View all tasks in a structured format  
- ✅ Mark tasks as completed  
- ❌ Delete tasks by ID  
- 📊 View task statistics (total, completed, pending, completion rate)  
- 💾 Automatic save & load using JSON file  

---

## 🛠️ Technologies Used
- 🐍 Python  
- 📦 JSON (data storage)  
- 📁 File Handling  

---

## ⚙️ How It Works
- Tasks are stored in a dictionary during runtime 🧠  
- Data is loaded from `data.json` at startup 🔄  
- All changes are saved back to the file on exit 💾
